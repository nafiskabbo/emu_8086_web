# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a vulnerability

emu8086web runs entirely in the browser and does not process secrets server-side. If you discover a security issue (XSS via shared links, dependency CVE, etc.), please open a private GitHub security advisory or email the maintainer.

Do not open a public issue for unpatched vulnerabilities.
