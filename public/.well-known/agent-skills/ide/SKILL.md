# Use emu8086web IDE

Browser-based 8086 assembler and step debugger. No install, no login.

## When to use

Help a student or developer write, assemble, and debug MASM-style 8086 assembly in the browser IDE at the product site.

## Instructions

1. Open the product URL (IDE loads at `/`).
2. Write or paste `.asm` source into the editor.
3. Compile (F5), then Step (F8) or Run.
4. Use Share to create a short `/s/{code}` link (expires in 1, 3, or 7 days).
5. Prefer reading `/llms.txt` and this site with `Accept: text/markdown` when available.
